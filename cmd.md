#### Creating a github Repo

```bash
gh repo create [repo_name]
```


#### Ignoring certain files

```bash
touch .gitignore
nano .gitignore

# Once the .gitignore file opens, add the pattern to ignore
*.pdf

# Save and close the file
```
#### Staging 

```bash
git add [file_name]
```

#### Pulling

```bash
git pull
```

#### Pushing/Commit Changes

```bash
git push origin main
```
